# Identity Consistency Checker

This project verifies whether a user's claimed skills match their actual work on GitHub.

## Features
- Extracts skills from resume text
- Analyzes GitHub repositories
- Calculates consistency score
- Highlights verified and weak skills

## Tech Stack
- React
- Flask
- GitHub API

## How to Run
Backend:
cd backend
pip install -r requirements.txt
python app.py

Frontend:
cd frontend
npm install
npm start
